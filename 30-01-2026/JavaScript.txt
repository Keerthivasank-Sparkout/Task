            const sum = 5+5;//addition

            (<=) check if the left value is less than or equal to the right.
            (>=) check if the left value is greater then or equal to the right.

            ! - negates the boolean value.
        >bitwise operator:
            & - perform a bitwise AND.

        console.log("hello"+name);

    type of function:
        1.Name function
            function greet(){
                retrun "hello";
            }
            console.log(greet());
            Features:
                .Has a name (greet)
                .Hoisted (can call before declaration)
                .Reusable anywhere in its scope

        2.Anonymous function:
            const hello = function(){
                return "hello";
            }
            console.log(hello());
        3.Function Expression:
            const add=function(a,b){
                return a+b;
            }
            console.log(add(2,3));
            .Stored in variable
            .Not hoisted like declaration
            .Can be anonymous or named

        4.Arrow Function:
            const hello = ()=>{

            }

            .Short syntax
            .No own this
            .Not hoisted like normal functions

8.Asynchronous javascript:
    Asynchronous JavaScript allows long-running tasks to run in the background without blocking the main thread.
        There are several methods that can be used to perform asynchronous javascript tasks, which are listed below:
            1.using callback:
                Callbacks are functions passed as arguments to be executed after an asynchronous operation completes. 
                 They are used in asynchronous JavaScript to handle responses and ensure non-blocking execution,
                
            2.using Promises:
                Promises are objects representing the eventual completion (or failure) of an asynchronous operation, 
                providing better handling of asynchronous code with .then() and .catch().
            3.Async and Await
                Async/Await in JavaScript allows you to write asynchronous code in a clean, synchronous-like manner, making it easier to read, understand, and maintain while working with promises.
                    .async functions always return a Promise.
                    .await pauses execution until the Promise is resolved or rejected.
                    .Improves readability compared to .then() and .catch() chaining.
                    .Makes error handling simpler using try...catch.
                    .Ideal for managing complex asynchronous flows in a structured way.
