1.What is Tailwind css?
    .Tailwind css is a utility-first framework that helps developers design modern user interfaces directly within their markup.
    .Instead of writing custom css for every component,Tailwind provides small,reusable utility classes thhat control layout,spacing,colors,typography and more.
    .This approach makes styling faster more consistent and easier to maintain.

2.color:
    .Tailwind css uses text-* utility classes to control text color.
    .All Style are written directlyinside the HTML.

    <p className="text-red-500"> Hello worlde</p> // text-red-500 used to chabge the text color to red and 500 is the opacity shade.
    <p classNam="text-[#hexacode]"> Hello world </p> used to give hexadecimal code to the text color.

3.margin and padding
    .padding
       In Tailwind CSS, padding is the space inside an element, between its content and its border. Tailwind provides utility classes to add padding in a simple and consistent way.

            <div class="bg-green-200">
            <p class="text-white p-4">Hello World</p>
            </div>
    .margin
        Margin in Tailwind CSS is the space outside an element, separating it from other elements. The classes mirror padding utilities:
        
            <div class="m-4 p-6 bg-blue-200">
            <p class="text-white">Content with margin and padding</p>
            </div>
4.typography:
    Typography utilities in Tailwind CSS allow you to control text appearance directly from HTML. This includes font size, weight, family, alignment, spacing, and responsive behavior without writing custom CSS.
        .Font Size
            Font size utilities help scale text consistently across the UI.

                <p class="text-xs">Extra small text</p>
                <p class="text-sm">Small text</p>
                <p class="text-base">Base text (default)</p>
                <p class="text-lg">Large text</p>
                <p class="text-xl">Extra large text</p>
                <p class="text-2xl">2XL text</p>
                <p class="text-3xl">3XL text</p>
                <p class="text-4xl">4XL text</p>
        .Font Weight
            Font weight controls how bold or light the text appears.

                <p class="font-thin">Thin text</p>
                <p class="font-light">Light text</p>
                <p class="font-normal">Normal text</p>
                <p class="font-medium">Medium text</p>
                <p class="font-semibold">Semi-bold text</p>
                <p class="font-bold">Bold text</p>
                <p class="font-extrabold">Extra bold text</p>
        .Font Family
            Tailwind provides font family utilities for different content types.

                Use cases:
                    .sans → UI and application text
                    .serif → Articles and reading-focused content
                    .mono → Code and technical content
                
                <p class="font-sans">Sans font (default)</p>
                <p class="font-serif">Serif font for articles</p>
                <p class="font-mono">Monospace font for code</p>
        .Text Alignment
            Text alignment utilities control how text is positioned inside its container.

                <p class="text-left">Left aligned text</p>
                <p class="text-center">Center aligned text</p>
                <p class="text-right">Right aligned text</p>
                <p class="text-justify">
                Justified text spreads evenly across the container width.
                </p>
        .Line Height (Leading)
            Line height affects readability by adjusting vertical spacing between lines.
                
                <p class="leading-none">No extra line spacing</p>
                <p class="leading-tight">Tight line spacing</p>
                <p class="leading-snug">Snug line spacing</p>
                <p class="leading-normal">Normal line spacing</p>
                <p class="leading-relaxed">Relaxed line spacing</p>
                <p class="leading-loose">Loose line spacing</p>
        .Letter Spacing (Tracking)
            Letter spacing controls the space between characters.
        
                <p class="tracking-tighter">Tighter letter spacing</p>
                <p class="tracking-tight">Tight letter spacing</p>
                <p class="tracking-normal">Normal letter spacing</p>
                <p class="tracking-wide">Wide letter spacing</p>
                <p class="tracking-wider">Wider letter spacing</p>
                <p class="tracking-widest">Widest letter spacing</p>
        .Text Transform
            Text transform utilities change text casing.
            
                <p class="uppercase">uppercase text</p>
                <p class="lowercase">LOWERCASE TEXT</p>
                <p class="capitalize">capitalize each word</p>
                <p class="normal-case">Normal Case Text</p>
        .Text Decoration
            Text decoration utilities add visual emphasis.
                
                <p class="underline">Underlined text</p>
                <p class="line-through">Line through text</p>
                <p class="no-underline">No underline applied</p>
        .Responsive Typography
            Responsive prefixes allow typography to adapt across screen sizes.
                
                <h1 class="text-xl md:text-3xl lg:text-5xl font-bold">
                Responsive Heading
                </h1>
5.Border:
    Borders play an important role in defining structure and visual separation in UI design. Tailwind CSS provides simple utility classes to control border width, color, style, and radius directly in HTML.
        .Border Width
            Border width utilities control how thick the border appears around an element. You can start with basic values and increase thickness as needed.
                <div class="border p-4">Default border</div>
                <div class="border-2 p-4">Border width 2</div>
                <div class="border-4 p-4">Border width 4</div>
                <div class="border-8 p-4">Border width 8</div>
        .Side-specific Borders
            You can apply borders to specific sides only.
                <div class="border-t p-4">Top border</div>
                <div class="border-b p-4">Bottom border</div>
                <div class="border-l p-4">Left border</div>
                <div class="border-r p-4">Right border</div>
        .Border Color
            Border color utilities let you match borders with your design system colors.
                <div class="border border-gray-300 p-4">Gray border</div>
                <div class="border border-red-500 p-4">Red border</div>
                <div class="border border-blue-600 p-4">Blue border</div>
        .Border Style
            Border style utilities control how the border line is drawn.                
                <div class="border border-solid p-4">Solid border</div>
                <div class="border border-dashed p-4">Dashed border</div>
                <div class="border border-dotted p-4">Dotted border</div>
                <div class="border border-double p-4">Double border</div>
                <div class="border-none p-4">No border</div>
6.Display and overflow:
    Block:
        Element takes full width, starts on new line.       
            <span class="block bg-blue-200">
            I am block
            </span>
        Even span behaves like div.
    inline-block:
        Only takes content width but allows width/height.
            <span class="inline-block w-32 h-12 bg-green-200">
            inline-block
            </span>
    inline:
        No width/height allowed.
            <span class="inline w-32 h-12 bg-red-200">
            inline
            </span>
        Width & height are ignored.
    hidden:
        Completely removes element from layout.
            <div class="hidden bg-gray-200">
            You can't see me
            </div>
    overflow-hidden
        Cuts off extra content.
            <div class="w-32 h-16 overflow-hidden border">
            This text is too long and will be clipped.
            </div>
    overflow-auto
        Scrollbars appear only when needed.
            <div class="w-32 h-16 overflow-auto border">
            Long content here that causes scrolling when required.
            </div>
    overflow-scroll
        Scrollbars always visible.
            <div class="w-32 h-16 overflow-scroll border">
            Long content here with forced scrollbars.
            </div>
    Direction-Specific Overflow
        Control X and Y separately.
            <div class="w-32 h-16 overflow-x-scroll overflow-y-hidden border">
            Horizontal scrolling content only
            </div>
7.Flex:
    Flexbox utilities in Tailwind CSS make it easy to build flexible, responsive layouts. By applying simple utility classes, you can control direction, alignment, spacing, wrapping, 
    and ordering of elements directly in HTML.
        .flex (Enable Flexbox)
            .The flex class enables Flexbox on a container. All direct children become flex items.
                <div class="flex border">
                <div class="p-2 bg-blue-200">Item 1</div>
                <div class="p-2 bg-blue-300">Item 2</div>
                </div>
            .Items are placed in a row by default
            .Flexbox works on the main axis (horizontal by default)
        .flex-col
            Items are arranged vertically from top to bottom.
                <div class="flex flex-col">
                <div>One</div>
                <div>Two</div>
                </div>
        .Justify Content (Main Axis)
            Controls alignment along the main axis.
                <div class="flex justify-start">...</div>
                <div class="flex justify-center">...</div>
                <div class="flex justify-end">...</div>
                <div class="flex justify-between">...</div>
                <div class="flex justify-around">...</div>
                <div class="flex justify-evenly">...</div>
        .Align Items (Cross Axis)
            Controls alignment along the cross axis.
                <div class="flex items-start">...</div>
                <div class="flex items-center">...</div>
                <div class="flex items-end">...</div>
                <div class="flex items-stretch">...</div>
        .Gap Between Items
            Gap utilities add spacing between flex items without using margins.
                <div class="flex gap-2">
                <div class="bg-pink-200 p-2">1</div>
                <div class="bg-pink-300 p-2">2</div>
                </div>
        .Flex Wrap
            No Wrap (default)
                Items stay on a single line even if they overflow.
                    <div class="flex flex-nowrap">
                    ...
                    </div>
            Wrap
                Items move to the next line when space runs out.
                    <div class="flex flex-wrap">
                    <div class="w-32 bg-blue-200">1</div>
                    <div class="w-32 bg-blue-300">2</div>
                    <div class="w-32 bg-blue-400">3</div>
                    </div>
8.Grid:
    .Grid utilities in Tailwind CSS allow you to build powerful two-dimensional layouts — controlling both rows and columns at the same time. 
    While Flexbox works in one direction (row or column), Grid lets you design full page layouts, dashboards, and complex UI structures easily.
    .Flexbox is one-dimensional. Grid is two-dimensional.
        .grid (Enable Grid)
            The grid class turns a container into a CSS Grid container. All direct children become grid items.
                <div class="grid border gap-2">
                <div class="bg-blue-200 p-2">Item 1</div>
                <div class="bg-blue-300 p-2">Item 2</div>
                </div>
            .Items are placed into grid cells 
            .Layout can be controlled using rows and columns
        .Grid Columns (grid-cols-*)
            These classes define how many columns your grid should have.
                <div class="grid grid-cols-2 gap-2">
                <div class="bg-green-200">1</div>
                <div class="bg-green-300">2</div>
                </div>

                <div class="grid grid-cols-3 gap-2">
                <div class="bg-green-200">1</div>
                <div class="bg-green-300">2</div>
                <div class="bg-green-400">3</div>
                </div>
            • grid-cols-2 → 2 equal-width columns • grid-cols-3 → 3 equal-width columns 
            • Items automatically flow into the next column
        .Grid Rows (grid-rows-*)
            You can also define how many rows the grid should have.
                <div class="grid grid-rows-3 gap-2 h-48">
                <div class="bg-pink-200">Row 1</div>
                <div class="bg-pink-300">Row 2</div>
                <div class="bg-pink-400">Row 3</div>
                </div>
            • grid-rows-3 creates three equal-height rows 
            • Height must be set (like h-48) to see row sizing
        .Gap (Spacing Between Cells)
            Same gap everywhere
                <div class="grid grid-cols-3 gap-4">
                ...
                </div>
            Different row & column gaps
                <div class="grid grid-cols-3 gap-x-6 gap-y-2">
                ...
                </div>
        .Column Span (col-span-*)
            Allows a grid item to stretch across multiple columns.
                <div class="grid grid-cols-4 gap-2">
                <div class="col-span-2 bg-yellow-200">Span 2</div>
                <div class="bg-yellow-300">1</div>
                <div class="bg-yellow-400">1</div>
                </div>
            • col-span-2 makes the item occupy two columns 
            • Useful for headers, featured cards, and wide content
        .Row Span (row-span-*)
            Allows a grid item to stretch across multiple rows.
                <div class="grid grid-rows-3 grid-cols-3 gap-2 h-48">
                <div class="row-span-2 bg-purple-200">Tall</div>
                <div class="bg-purple-300">1</div>
                <div class="bg-purple-400">2</div>
                </div>
            • row-span-2 makes the item twice as tall 
            • Great for featured cards and side content
        .Responsive Grid (Very Important)
            Grid becomes extremely powerful when combined with responsive breakpoints.
                <div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 gap-4">
                <div class="bg-gray-200">1</div>
                <div class="bg-gray-300">2</div>
                <div class="bg-gray-400">3</div>
                </div>
            • Mobile → 1 column • Tablet → 2 columns 
            • Desktop → 3 columns
            Perfect for cards, product grids, and galleries.
9.Interactivity & Effects in Tailwind CSS
    Hover, Focus & State
        Tailwind uses modifiers like hover:, focus:, and active: to apply styles only when a specific state is active.
            hover:
                Changes the style when the mouse is over the element.
                    <button class="bg-blue-500 hover:bg-blue-600 text-white px-4 py-2 rounded">
                    Hover Me
                    </button>
            focus:
                Applies styles when an element (like an input) is selected or receives focus.
                    <input class="border border-gray-300 px-3 py-2 rounded focus:outline-none focus:ring-2 focus:ring-blue-500" placeholder="Focus me...">
            active:
                Applies styles while the element is being clicked (pressed down).
                    <button class="bg-green-500 text-white px-4 py-2 rounded active:scale-95 transition">
                    Click Me
                    </button>
            group-hover:
                Allows a child element to change styles when its parent is hovered. You must add the group class to the parent.
                    <div class="group border p-4 rounded hover:bg-gray-100 cursor-pointer">
                    <p class="text-gray-700 group-hover:text-red-500 font-bold">
                    Hover the box to change my color!
                    </p>
                    </div>
    Transitions & Motion
        Tailwind makes it easy to animate changes smoothly using transition utilities.
            transition
                Enables smooth interpolation between property values.
                    <div class="bg-purple-500 text-white p-4 rounded transition hover:bg-purple-700">
                    Smooth Background Change
                    </div>
            duration-*
                Controls the speed of the transition in milliseconds.


                    <button class="bg-blue-500 text-white px-4 py-2 rounded transition duration-300 hover:scale-105">
                    300ms Duration
                    </button>

                    <button class="bg-blue-500 text-white px-4 py-2 rounded transition duration-1000 hover:scale-105 ml-2">
                    1000ms Duration
                    </button>
            ease-*
                Controls the pace of the animation (e.g., ease-in, ease-out, ease-in-out).
                    <div class="transition ease-in-out duration-500 transform hover:translate-x-4 bg-orange-400 p-2 inline-block rounded">
                    Ease In Out
                    </div>
            scale-*
                Zooms an element in or out. Often used with hover: for interaction feedback.
                        <div class="bg-tea-500 p-4 bg-indigo-500 text-white rounded transition hover:scale-110 inline-block">
                        Zoom on Hover
                        </div>

    Shadows & Depth
        Shadows give depth to your UI, making elements stand out like cards or modals.
            shadow-*
                Adds a box-shadow. Levels range from shadow-sm to shadow-2xl.
                    <div class="shadow-md hover:shadow-xl p-6 bg-white rounded transition duration-300">
                    <h3 class="font-bold">Card with Shadow</h3>
                    <p class="text-gray-600">Hover to elevate this card.</p>
                    </div>
    Opacity & Blur
        Control transparency and background blurring for modern layout effects.
            opacity-*
                Sets the transparency of an element. Ranges from 0 to 100.
                    <div class="bg-black text-white p-2 opacity-50">
                    50% Opacity
                    </div>
                    <div class="bg-black text-white p-2 opacity-100 mt-2">
                    100% Opacity (Default)
                    </div>
            backdrop-blur
                Blurs the content behind an element. Useful for overlays or "glassmorphism" effects. Note: Requires a semi-transparent background color (like bg-white/30) to be visible.
                    <div class="relative w-full h-32 bg-url('https://source.unsplash.com/random') bg-cover">
                    <div class="absolute inset-0 flex items-center justify-center">
                    <!-- Glass Effect -->
                    <div class="backdrop-blur-md bg-white/30 border border-white/20 p-4 rounded shadow-lg">
                        <p class="font-bold text-black">Glass Effect</p>
                    </div>
                    </div>
                    </div>
